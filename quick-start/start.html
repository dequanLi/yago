Hello World

创建你的hello world

0.1. 用 yago 在当前目录创建你的项目 my_app

yago init -a my_app

0.2. 进入目录初始化mod

cd my_app/
go mod init

0.3. 构建

go build

如 go build 遇报错，请看解决方案

0.4. 创建属于自己的配置文件，并启动

sh env.init.sh yourname
./my_app
